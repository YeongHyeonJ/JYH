git
프로젝트 진행시 유지보수를 편하게 해주는 것
수정된 이력이 보임
명령어를 이용해서 동작

github 회원가입 및 설치
온라인 저장소를 무료로 제공
githun.com 접속 및 회원가입
구글에 github desktop 검색 후 운영체제에 맞게 다운로드
.gitignore
 git에 업로드할 때 업로드 하고싶지 않은 파일들을 저어놓음
 대부분 확장자나 확정폴더를 지정, 간혹 특정 파일을 지정함.

clone
온라인 저장소에 있는 파일들을 내 PC로 복사

commit
내 PC에서 수정된 내용을 기록

push
내 PC에 수정된 이력들을 온라인 저장소에 업데이트

pull
온라인 저장소에 있는 내용으로 내PC를 업데이트
